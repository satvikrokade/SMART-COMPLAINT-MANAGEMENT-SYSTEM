**Smart Complaint System – A Digital Solution for Municipal Services**

Built a web platform that allows citizens to report civic issues, track their status, and stay updated in real time — while authorities can efficiently manage and resolve complaints.

 Focused on improving transparency, reducing delays, and creating a better connection between citizens and municipal departments.

⚙️ Tech Stack: React, Node.js, Express, MySQL, Cloudflare
